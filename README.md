# BR Client 


Custom electron client for popular battle royale io game: BuildRoyale.io ( Inspired of Delta Client ) 

## Current features
  - Auto updater
  - Logging in
  - Mac Support

## Planned features
  - DiscordRPC
  - FPS meter
  - Linux support
  - Adblocker
  - Custom Mods
  - And more!
